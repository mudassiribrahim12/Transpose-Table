Important File Instructions:
When opening downloaded Excel files, you may see a warning: "The file format and extension of '[transposed_data_XXX.xls]' don't match. The file could be corrupted or unsafe. Unless you trust its source, don't open it. Do you want to open it anyway?"

To fix this:
  1. Click "Yes" to open the file anyway
  2. Click File → Save As
  3. Choose a name and select "Excel Workbook (.xlsx)" as the format
  4. Click Save - it will work perfectly!
Note: The same applies to CSV files if you see similar warnings.
